### 关于改变this
new Function、eval、setTimeout 哪个能改变this？

eg：
``` js
obj = {
    name: 'name is obj',
    sayName() {
        new Function('console.log(this)')(); // window
        setTimeout('console.log(this)', 0); // window 另：CSP下报错
        eval('console.log(this)'); // obj
    }
};

obj.sayName();
```
